## Running Locally

Clone the project

```
git clone https://github.com/Greenaustin2/portfolio_website.git
```

Go to project directory

```
cd portfolio
```

Remove remote origin

```
git remote remove origin
```

Install dependencies

```
npm install
```

Start the server

```
npm start
```
